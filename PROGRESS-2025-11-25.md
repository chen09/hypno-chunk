## 🔧 技术细节

### JSON 格式支持
项目现在支持两种 JSON 格式：
1. **语义模块格式**（Type 1）: 用于学习单词、词组、习语等，包含完整的语义信息
2. **常用表达式格式**（Type 2）: 用于学习常用英语短句，仅包含英文句子和中文翻译

### 音频生成流程
1. 合并所有 part JSON 文件 → `*_merged.json`
2. 翻译检查（如有需要）
3. 生成音频文件（英文慢速 + 中文翻译 + 英文快速）
4. 导出完整音频和字幕文件
